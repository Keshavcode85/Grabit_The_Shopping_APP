# 🛍️ Grabit – The Shopping App

Grabit is a modern and responsive e-commerce web application built using HTML, CSS, and Vanilla JavaScript.  
It fetches live product data from the FakeStore API and provides a seamless shopping experience with dynamic cart management.

## ✨ Features

- 📦 Fetch products from external API
- 🛒 Add to Cart functionality
- ❌ Remove items from cart
- 💾 Cart data stored using LocalStorage
- 🔄 Real-time cart updates
- 📱 Fully responsive design
- ⚡ Dynamic DOM manipulation

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- FakeStore API
